TCP-IP协议详解

2019.02.25 Grooter 计算机网络  热度 ℃

协议分层之OSI参考模型

  1. 应用层

    负责各应用程序的通信，如QQ

  2. 表示层

    负责将不规范的数据统一转换为国际标准，预防数据乱码，如utf-8

  3. 会话层

    决定何时建立连接，以及数据的传播顺序

  4. 传输层

    建立和断开数据传输连接，保证可靠传输，检查是否有数据丢失现象

  5. 网络层

    寻找传播路径，如路由信息

  6. 数据链路层

    数据帧与比特流之间的转换，负责某条链路的数据传输

  7. 物理层

    高低电位与比特流之间的转换，负责指定连接器与网线的规格

传输方式分类

  • 有连接

  • 无连接

  • 电流交换

  • 分组交换

  • 单播

  • 广播

  • 任播
